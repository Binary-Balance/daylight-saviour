# Time-Zone Data

`@daylight-saviour/time-zone-data` owns pinned IANA-derived inputs, deterministic
generation, and bundled offline Time-Zone Data Packs.

Australian Coverage uses tzdb 2026c archive SHA-256
`e4a178a4477f3d0ea77cc31828ff72aa38feff8d61aa13e7e99e142e9d902be4`.
Evidence comes from `antarctica`, `asia`, `australasia`, `backward`,
`zone.tab`, and `zone1970.tab`. Exact rule inputs, 18 zone configurations,
coverage assumption, and fixed generation instant are reviewable in
`source/tzdb-2026c-australian-coverage.json`.

Generator interprets IANA standard-time (`s`) and wall-time transition bases
with UTC calendar arithmetic. Lord Howe therefore retains its 30-minute wall
change semantics. Generator never asks host ICU, host zoneinfo, or current
clock for civil-time rules.

Run `npm run generate --workspace @daylight-saviour/time-zone-data` after a
reviewed source/configuration change. Tests require regenerated bytes to match
committed known-good pack exactly.

## Signed artifact publisher

`scripts/sign-australian-pack.mjs` reads private key material only from an
explicit file, validates pack content, signs exact committed bytes with Node
Ed25519, and emits deterministic artifacts:

```sh
output_directory="$(mktemp -d)"
node packages/time-zone-data/scripts/sign-australian-pack.mjs \
  --private-key packages/time-zone-data/test-fixtures/TEST-ONLY-ed25519-private.pem \
  --key-id test-only-2026-a \
  --output-directory "$output_directory"
```

Committed fixture key is public test material and must never sign candidate or
production data. Real signing inputs, publisher identity, artifact location,
and promotion workflow belong outside this public repository.

`scripts/serve-test-deployment.mjs` is a loopback reference deployment. It
serves manifest with ETag/Last-Modified validators and `no-cache`, while
versioned packs use one-year immutable caching:

```sh
node packages/time-zone-data/scripts/serve-test-deployment.mjs \
  --directory "$output_directory" \
  --port 8088
```

`tests/test-deployment.test.mjs` starts harness on ephemeral loopback port and
proves 200/304 behavior plus exact immutable artifact bytes. It does not mutate
cloud or private environments.
