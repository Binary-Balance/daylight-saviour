# Use a bureaucratic-modernist visual language

Daylight Saviour will combine premium editorial composition with restrained Australian bureaucratic cues. Oversized time and status typography, a disciplined grid, subtle document marks, and dry status stamps will provide character without a mascot or novelty-app finish.

## Consequences

- Light presentation centres warm daylight cream and dark presentation centres midnight navy, with one sharp semantic accent refined during visual design.
- Typography and spacing establish hierarchy before decoration; critical facts remain immediately scannable.
- Forward and Backward Changes may temporarily distort, skip, reverse, or echo grid and type while reduced-motion mode preserves the same meaning without kinetic effects.
- Official-document motifs remain subtle and fictional; the interface must not imply government affiliation.
- Cartoon clocks, generic gradient-led styling, flags, kangaroos, and other obvious Australian clichés are excluded.
- Final tokens, components, and motion recipes must work in both OS-controlled themes and pass accessibility release gates.
